# Pilates Studio Manager

Aplicativo web para testar a rotina de gestao de um studio de pilates: alunos, turmas, presencas, evolucoes, pagamentos, planos e despesas.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Observacao sobre dados

Nesta versao de testes, os dados ficam salvos no `localStorage` do navegador. Isso ajuda a testar durante a semana sem backend, mas os dados ainda ficam apenas no dispositivo usado.
